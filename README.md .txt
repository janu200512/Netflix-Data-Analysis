# Netflix Data Analysis

## Project Overview
This project analyzes Netflix movies and TV shows dataset to extract insights and visualize content trends.

## Dataset
The dataset contains information about Netflix titles including type, release year, genre, country, rating, and description.  
Download from Kaggle: [Netflix Movies and TV Shows Dataset](https://www.kaggle.com/datasets/shivamb/netflix-shows)

## Folder Structure
- `data/` : Dataset files
- `notebooks/` : Jupyter notebooks for analysis
- `scripts/` : Data cleaning or preprocessing scripts
- `reports/` : Analysis report

## How to Run
1. Install required packages:

2. Open `notebooks/netflix_analysis.ipynb` and run the notebook.

## Key Analyses
- Content type distribution (Movies vs TV Shows)
- Genre and country distribution
- Trends over time
- Ratings analysis
